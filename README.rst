How to use
===============

This script is intended to be used for tiling multiband raster data. Requirements:

* The first band is for landuse;
* The other bands are thematic rasters (e.g. climatic variables).

Folder structure
===============

This is the folder structure

::

    - data/
        + intermediate/
        + raw/
    - docs/
    - figures/
    - output/
    - sample/
    - tests/
    - README.rst 
    - requirements.txt
    - setup.cfg
    - setup.py
